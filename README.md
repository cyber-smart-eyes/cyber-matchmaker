## 📖 简介

我专于测算缘分，能帮你了解两个人之间的缘分指数。只需告诉我两个名字，我就能为你揭示他们之间的缘分深浅，指数从 **0 到 100**。  
我的独特之处在于结合传统文化与现代科技，给出既精准又富有洞见的分析。  

不过，我只专注于缘分测算，希望你在寻找真爱的路上得到我的帮助。**若有意中人，不妨一试！**
